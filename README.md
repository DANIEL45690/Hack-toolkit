## 🚀 Возможности

### 🕵️ OSINT и DOX-инструменты

| Модуль | Описание |
|--------|----------|
| IP-адрес | WHOIS, геолокация, проверка VPN, репутация, история |
| Домен | WHOIS, DNS-записи, хостинг |
| Email | Поиск утечек (HIBP), MX-записи |
| Username | Поиск на 20+ платформах (GitHub, Twitter, Instagram и др.) |
| Телефон | Определение страны, оператора, координат |
| Криптокошелёк | Баланс и транзакции BTC / ETH |
| Поиск веб-камер | По стране/городу |
| Google Maps | Прямой переход по координатам |

### ⚡ Атаки и нагрузочное тестирование

- **DDoS (HTTP Flood)** — многопоточная отправка GET-запросов
- **IP DoS (TCP Flood)** — подключения к указанному порту
- **SMS Bomber** — массовая отправка запросов к Telegram-сервисам

### 🔍 Сканеры и рекон

- **Порт-сканер** (отдельные порты или диапазоны до 65535)
- **Сканер поддоменов** (из встроенного словаря)
- **DNS-рекон** (A, MX, NS, TXT, CNAME и др.)
- **SSL-проверка** (срок действия, issuer, SAN)
- **Reverse IP** — поиск доменов на одном IP

### 🧰 Веб-безопасность

- **Детектор технологий** (CMS, фреймворки, сервера)
- **Проверка security-заголовков** (HSTS, CSP, X-Frame-Options)
- **Поиск уязвимостей** (SQLi, XSS, LFI, CSRF и др.)
- **CVE-поиск** (по ключевому слову)
- **Wayback Machine** — исторические снимки сайтов

### 🔐 Брутфорс

- **Gmail SMTP** — перебор паролей
- **Instagram** — авторизация по паролям (с задержками)

### ⚙️ Утилиты

- **Генератор паролей** (с оценкой стойкости)
- **Хэш-калькулятор** (MD5, SHA1/256/512, SHA3, BLAKE2)
- **Системный монитор** (CPU, RAM, Disk в реальном времени)
- **QR-генератор** (подготовлен, требует `qrcode`)
